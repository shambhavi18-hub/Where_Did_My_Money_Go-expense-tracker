# expense tracker V1:

A simple command-line expense tracker built in Python.

## Features
- Add expenses
- View all expenses
- Calculate total spending
- Handles empty expense lists gracefully
- Menu-driven interface

## Concepts Used
- Functions
- Lists
- Dictionaries
- Loops
- Conditionals
- User Input
- Data Structures
